# LOAM-multi-thread
Adjusted the original LOAM to a multi-threaded version which doesn't require ROS, so we can run the algorithm under multilple 
platforms which are not convenient with ROS like MacOS and Windows.

## Prerequisites
**PCL** \\
**Pangolin** \\
**Eigen** \\
